# 🚀 Ayush Kumar - Portfolio Website

A modern, responsive, and interactive developer portfolio showcasing my expertise in AI, ERPNext, Industrial Automation, Cloud Technologies, and Full-Stack Development.

This portfolio highlights real-world projects, case studies, technical skills, and professional experience through an elegant user experience inspired by modern SaaS and developer portfolios.

## ✨ Features

- 🎨 Modern dark/light theme with smooth transitions
- ⚡ Fully responsive design for desktop, tablet, and mobile
- 🔍 Command palette (⌘K / Ctrl+K) for quick navigation
- 📈 Animated statistics and scroll-based interactions
- 💼 Detailed project case studies with business impact
- 🤖 AI, ERPNext, and Industrial Automation showcase
- 🌟 Interactive UI with cursor glow and micro-animations
- 📄 Resume download support
- 📬 Contact section for professional inquiries
